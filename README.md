# 기능 구현 목록

---

## 기능 작동 흐름

- 경주 할 자동차 이름 입력 (쉼표로 이름 구분, 자동차 이름은 5자 이하)
- 시도할 횟수 입력 (n이라면 n번 작동)
- 각 시도할 때마다 자동차 별로 1~9의 수를 랜덤 생성한다.
  - 4이상이라면 전진
- 각 턴이 종료될 때마다 현재 상황 출력
- 모든 시도가 끝났을 때 우승자 출력 (공동 우승자 있을 수 있다.)

---

## 패키지 별 기능 구현 목록

- Domain
  - Car
    - 자동차의 이름과 위치를 관리한다.
  - Game
    - 게임의 턴을 관리하고 우승자 관리를 한다.

- Controller
  - GameController
    - 게임의 전반적인 흐름을 관리한다.

- Exception
  - CarException
    - 자동차 이름에 대한 예외 처리
  - GameException
    - 시도할 횟수 예외처리

- View
  - OutputView
    - 게임의 출력 담당
  - InputView
    - 게임의 입력 담당
